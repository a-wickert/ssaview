ssaview
-------
http://golang-ssaview.herokuapp.com/

ssaview is a small utlity that renders SSA code alongside input Go code

Runs via HTTP on :8080

License: ISC

```sh
  $ go get github.com/tmc/ssaview
  $ ssaview &
  $ open http://localhost:8080/
```
